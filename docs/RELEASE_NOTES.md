## 2. Current Draft Release

### SurveyIQ MVP - Draft

**Release Date:** TBD  
**Status:** Draft

#### Summary

SurveyIQ MVP enables users to upload survey feedback files, preview data, select a free-text feedback column, run AI-powered analysis, and export both analysed CSV files and management PDF reports.

#### New Capabilities

- Upload CSV or Excel files.
- Preview the first 10 rows.
- Detect column names.
- Select the feedback column.
- Analyse responses using AI.
- Generate themes, sentiment, confidence scores, and reasons.
- Generate executive insights and recommended actions.
- Download analysed CSV output.
- Download a professional management PDF report.

#### Improvements

- Batched AI analysis for larger files.
- Canonical theme consolidation.
- PDF report sections for executive summary, sentiment, themes, highlights, risks, actions, conclusions, and charts.

#### Known Limitations

- Local file storage only.
- No authentication yet.
- No database-backed project library yet.
- Larger analyses can take several minutes.
